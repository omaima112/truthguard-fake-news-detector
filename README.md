# truthguard-fake-news-detector
**TruthGuard** is a browser-based fake news detection web app that identifies misinformation using a custom logic and pattern recognition system—built fully with HTML, CSS, and JavaScript.  
No AI models or APIs—just real, working logic.

## 💡 How It Works

TruthGuard checks input text against a set of common misinformation **patterns** using conditional logic. If suspicious patterns are found, it flags the news as **potentially false**; otherwise, it marks it as **likely safe**.

This is an educational project to promote **media literacy** and teach how pattern-based detection can help filter misleading information.
